Meme SMP Season One was a weird season. Meme SMP Season One had a lot to live upto because of Meme SMP Season Zero. People had a lot of fun in Season Zero so when I started work on Season One I knew I had to keep the hype going so I did a few diffrent things. I self-hosted this season, Meaning I hosted the Server on my own PC. I also spent months going through and finding plugins that would make the server better. I started the server on August 4, 2021 (It was delayed by about five days initally). It was subsequently shutdown on September 10th, 2021.
